# This function will conduct the chi-square test of independence.
# It will output frequency table, degree of freedom, chi2 test statistic, and p-value

def compute_chi_square(x, y):
    freqtab = pd.crosstab(x, y)
    print("Frequency table")
    print("============================")
    print(freqtab)
    print("============================")
    chi2, pval, dof, expected = sp.chi2_contingency(freqtab)
    print('degree of freedom: ', dof)
    print('Chi-Square test statistic: ', chi2)
    print('p-value: ', pval)
    return

Overall 75% of the female passengers survived the accident where only around 20% of the male passengers did. This could be due to two possible reasons: 1) ladies are given the priority to board the lifeboats; 2) there are twice as many men as there are ladies, so more men were left out of the lifeboats. If that was the case, we would expect the survival proportion of the two genders to reflect the percentage of the total passengers.

Looking at the second graph we see that out of all the survived passengers, 70% of them are females and 30% are male, which do not reflect the gender proportion of the total population, so we can conclude that ladies are given the priority to board the lifeboats.

# This function divides the ages into groups, junior means age smaller than 10

def convert_age(age):
    if age < 10.0:
        return "Junior"
    elif age < 20.0:
        return "10s"
    elif age < 30.0:
        return "20s"
    elif age < 40.0:
        return "30s"
    elif age < 50.0:
        return "40s"
    elif age < 60.0:
        return "50s"
    elif age < 70.0:
        return "60s"
    elif age < 80.0:
        return "70s"  
    elif age < 90.0:
        return "80s" 
    elif age < 100.0:
        return "90s" 
    else:
        return "NaN"

Summary¶

In conclusion, we find that sex, passenger class, family size, and embarkment location are all important factors impacting a passenger's survival chance.

Female passengers are more likely to survive than male passengers, this is probably due to the lady first rule when boarding lifeboats.

Class 1 passengers are the most likely to survive, and Class 2 passengers also have above average survival rate. We can conclude that passengers with higher class ranking are given the priority to board first.

While passengers travelling by themselves have an average chance of surviving, those with a normal sized family onboard (2-4 people) have high chances around 60% to 80%. This is probably due to people being sympathetic, not willing to break up families. However, larger size family (5-11 people) aren't so lucky, probably because it was difficult for the large group to maneuver.

Surprisingly, embarkment location also made a difference in passengers' survival. Cherbourg passengers had higher survival rate than passengers from the other two locations, but this probably because Cherbourg had higher percentage of Class 1 passengers.

Lastly, also to my surprise, age didn't play an important factor, since I always thought that children were given priorities to board lifeboats first.

Limitations¶

The most important limitation I could think of is the incompleteness of Age information. Since about 1/6 of the passengers don't have age recorded, the age analysis could be incorrect.

There are also several other factors I wish I could explore, such as race, nationality. Back in the day, the society was not as fair and just as it is today. Race and nationality could have played an important factor.
